# Atividade Proz - Frontend Login

Este projeto consiste em uma página de login moderna e responsiva, desenvolvida como parte de uma atividade de desenvolvimento frontend. A interface foca na experiência do usuário e na simplicidade.

## 🚀 Funcionalidades

- **Interface Responsiva**: Utiliza Bootstrap para garantir que a página funcione bem em dispositivos móveis e desktops.
- **Alternar Visibilidade de Senha**: Inclui um botão para mostrar ou ocultar a senha digitada.
- **Validação Básica**: Campos obrigatórios (`required`) e limites de caracteres definidos diretamente no HTML (`maxlength`).
- **Estilização Customizada**: Combinação de Bootstrap com estilos CSS personalizados.

## 🛠️ Tecnologias Utilizadas

- HTML5 - Estruturação.
- Bootstrap 5.3.8 - Framework CSS para layout e componentes.
- Bootstrap Icons - Biblioteca de ícones.
- JavaScript - Lógica de interação do formulário.

## 📂 Estrutura do Projeto

```text
atividade-proz-frontend-main/
├── index.html              # Página principal
└── src/
    └── assets/
        ├── scripts/
        │   └── login.js    # Lógica de exibição de senha e submissão
        └── styles/
            └── login.css   # Estilização complementar
```

## 🔧 Como Executar

1. Faça o download ou clone este repositório.
2. Navegue até a pasta raiz do projeto.
3. Abra o arquivo `index.html` em qualquer navegador web moderno.
